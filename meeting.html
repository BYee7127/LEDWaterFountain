  • Testing pumps with new power supply -- IT WORKS!!!!!!!!!
  • Testing LEDs with new power supply -- IT WORKS.
    • Logic shifter needs a wire connecting 3.3V power to pin OE to enable it.
    • Output to the LED comes out from pin B8.
    • Input from the Teensy comes in from pin A8.
      • This wire was shifted from A8 to OE, which initially caused the NeoPixel not to work.
  • Testing pumps and LEDs at the same time - IT WORKS
  • More FFT work
    • Need to adjust clock speed to 96MHz (recommended in Teensy documentation)
      • Solution
      • Add following code to sketch: 
        // outside and above setup
#if defined(__IMXRT1062__)
extern "C" uint32_t set_arm_clock(uint32_t frequency);
#endif

// inside setup
#if defined(__IMXRT1062__)
  set_arm_clock(24000000);
  Serial.print("F_CPU_ACTUAL=");
  Serial.println(F_CPU_ACTUAL);
#endif
    • FFT available function is returning false all the time.
